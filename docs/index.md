# Документация проекта zbx-1c-py

**zbx-1c-py** — это кроссплатформенный инструмент для интеграции 1С:Предприятия с системой мониторинга Zabbix. Проект позволяет собирать информацию о кластерах 1С, сессиях пользователей и фоновых заданиях через сервер администрирования кластера (RAC) и передавать эти данные в Zabbix для последующего мониторинга и анализа. Проект работает на Windows, Linux и macOS. Проект полностью изолирован от системного Python и использует uv для управления зависимостями и обеспечения воспроизводимости среды выполнения.

## Быстрый старт

### Установка
```bash
# Установка зависимостей
uv sync

# Проверка доступности RAS
uv run src/zbx_1c_py/main.py --check-ras

# Получение списка кластеров
uv run src/zbx_1c_py/main.py --discovery
```

### Основные режимы работы
- `--discovery`: Режим обнаружения кластеров для Zabbix LLD
- `--check-ras`: Проверка доступности RAS-сервиса
- `<cluster_id>`: Сбор метрик для конкретного кластера

Для более подробной информации см. соответствующие разделы документации.
